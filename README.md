# Claude Desktop Skills

Custom skills for Claude Desktop.

| Skill | Description | Download |
|-------|-------------|----------|
| [Resume Evaluator](skills/resume-evaluator/) | Score resumes, check ATS compatibility, rewrite bullets | [Download .skill](releases/resume-evaluator.skill) |

## Installation

1. Download the `.skill` file from releases
2. Open Claude Desktop → Settings → Skills
3. Click "Add Skill" and select the file
4. Done! Ask Claude to review your resume.
